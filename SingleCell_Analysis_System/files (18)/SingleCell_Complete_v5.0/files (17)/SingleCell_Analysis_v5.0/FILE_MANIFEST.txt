🔍 核心功能详解
================================================================================

Module 1: 核心分析（9个函数）
--------------------------------------------------------------------------------
1. read_and_process_10x_data()
   - 自动读取多个10X数据目录
   - 进度条显示
   - 断点续传支持

2. create_seurat_with_qc()
   - 创建Seurat对象
   - 质控过滤
   - 生成4个QC图表:
     * 01_QC_ViolinPlot.pdf
     * 02_QC_FeatureScatter.pdf
     * 03_QC_Feature_Density.pdf
     * 04_QC_Statistics_Comparison.pdf

3. normalize_and_reduce()
   - 数据归一化
   - 识别高变基因
   - PCA降维
   - UMAP可视化
   - 聚类分析
   - 生成4个图表:
     * 01_VariableFeatures.pdf
     * 02_PCA_Plot.pdf
     * 04_ElbowPlot.pdf
     * 01_UMAP_Clusters.pdf/png

4. find_all_markers()
   - 批处理Marker分析（内存优化）
   - 生成3个图表:
     * 01_Markers_Heatmap.pdf
     * 02_Markers_DotPlot.pdf
     * 03_Top_Markers_FeaturePlot.pdf
   - 保存CSV文件:
     * All_Markers.csv
     * Top10_Markers_Per_Cluster.csv

5. automatic_cell_annotation()
   - SingleR自动注释
   - 生成注释UMAP图
   - 细胞类型映射表

6. manual_cell_annotation()
   - 应用手动注释
   - 生成最终注释UMAP
   - 注释映射文件

7. perform_differential_expression()
   - 组间差异分析
   - 生成3个图表:
     * 01_Volcano_Plot_Enhanced.pdf
     * 02_MA_Plot.pdf
     * 03_Top_DiffGenes_FeaturePlot.pdf
   - Differential_Genes.csv

8. analyze_proportions()
   - 细胞比例分析
   - 生成4个图表:
     * 01_Proportions_Stacked.pdf
     * 02_Proportions_Grouped.pdf
     * 03_Proportions_Bubble.pdf
     * 04_Pie_Chart_*.pdf
   - Cell_Proportions.csv

9. save_module1_results()
   - 保存所有结果
   - Seurat.RData
   - 表达矩阵（采样5000细胞）
   - 元数据

高级可视化: jjVolcano + Gene Cluster Heatmap
--------------------------------------------------------------------------------
1. generate_jjVolcano_plot()
   - 多聚类faceted火山图
   - 自动标记top基因
   - 统计摘要

2. generate_gene_cluster_heatmap()
   - 基因聚类（kmeans/hclust）
   - GO富集分析
   - 集成热图展示
   - 标记已知marker

🎨 SCI级别可视化特色
================================================================================

配色方案:
----------------------------------------
所有图表采用柔和的SCI级别配色:
  - 主色调: 浅蓝(#E8F4F8) → 中蓝(#A8D8E8) → 深蓝(#7BA7BC)
  - 强调色: 珊瑚红(#D67280) 用于差异表达
  - 热图: 蓝-白-红渐变
  - 背景: 简洁的theme_minimal()

图表质量:
----------------------------------------
  - 分辨率: 600 DPI (PDF), 300 DPI (PNG)
  - 字体: 加粗标题，清晰轴标签
  - 尺寸: 适合发表的标准尺寸
  - 格式: PDF矢量图 + PNG位图
