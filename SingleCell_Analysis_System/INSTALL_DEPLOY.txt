⚙️ 安装步骤
================================================================================

步骤1: 安装R和RStudio
----------------------------------------
1. 下载并安装R:
   Windows/Mac: https://cran.r-project.org/
   Linux: sudo apt-get install r-base

2. 下载并安装RStudio (推荐):
   https://posit.co/download/rstudio-desktop/

步骤2: 安装基础R包
----------------------------------------
打开R或RStudio，运行以下命令:

# 设置CRAN镜像（中国用户推荐）
options(repos = c(CRAN = "https://mirrors.tuna.tsinghua.edu.cn/CRAN/"))

# 或使用官方镜像
options(repos = c(CRAN = "https://cloud.r-project.org/"))

# 安装基础包
install.packages(c(
  "Seurat", "dplyr", "ggplot2", "patchwork",
  "RColorBrewer", "ggrepel", "ggsci", "viridis",
  "tidyr", "stringr", "readr", "Matrix", "scales"
))

步骤3: 安装Bioconductor包
----------------------------------------
# 安装BiocManager
if (!requireNamespace("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

# 安装Bioconductor包
BiocManager::install(c(
  "SingleR", "celldex", "org.Hs.eg.db", 
  "clusterProfiler", "ComplexHeatmap"
))

步骤4: 安装高级可视化包（可选）
----------------------------------------
# jjVolcano所需
install.packages("scRNAtoolVis")

# Gene Cluster Heatmap所需
install.packages("remotes")
remotes::install_github("junjunlab/ClusterGVis")

注意: 如果GitHub访问困难，可以跳过高级可视化

步骤5: 验证安装
----------------------------------------
# 运行以下命令验证所有包是否安装成功
required_packages <- c(
  "Seurat", "dplyr", "ggplot2", "SingleR", 
  "celldex", "org.Hs.eg.db"
)

for (pkg in required_packages) {
  if (!require(pkg, character.only = TRUE, quietly = TRUE)) {
    cat(sprintf("❌ %s 未安装\n", pkg))
  } else {
    cat(sprintf("✅ %s 已安装\n", pkg))
  }
}

✅ 验证安装

运行验证脚本:
----------------------------------------
# 在R中运行
cat("=== 系统验证 ===\n")

# 1. 检查R版本
r_version <- R.version.string
cat(sprintf("R版本: %s\n", r_version))
if (as.numeric(R.version$major) >= 4) {
  cat("✅ R版本符合要求\n")
} else {
  cat("❌ R版本过低，请升级到4.0+\n")
}

# 2. 检查内存
if (.Platform$OS.type == "windows") {
  mem <- memory.limit()
  cat(sprintf("可用内存: %.1f GB\n", mem/1024))
}

# 3. 检查文件
files <- c(
  "Module1_Core_Analysis_ULTIMATE.R",
  "Module1_DiffAnalysis_Proportions.R",
  "Advanced_Visualizations.R",
  "Run_Complete_Analysis.R"
)

for (f in files) {
  if (file.exists(f)) {
    cat(sprintf("✅ %s\n", f))
  } else {
    cat(sprintf("❌ %s 缺失\n", f))
  }
}

# 4. 检查数据目录
if (dir.exists("data")) {
  cat("✅ data目录存在\n")
  data_files <- list.files("data", recursive = TRUE)
  cat(sprintf("   包含 %d 个文件\n", length(data_files)))
} else {
  cat("⚠️  data目录不存在，请创建并添加数据\n")
}

cat("\n如果所有检查都通过，可以开始分析！\n")

================================================================================
🐛 常见安装问题
================================================================================

问题1: 包安装失败
----------------------------------------
错误: "package XXX is not available"

解决方案:
1. 检查包名拼写
2. 更换CRAN镜像
3. 更新R版本
4. 使用管理员权限运行R

问题2: Bioconductor包安装失败
----------------------------------------
错误: BiocManager安装失败

解决方案:
# 手动设置Bioconductor镜像
options(BioC_mirror = "https://mirrors.tuna.tsinghua.edu.cn/bioconductor")
BiocManager::install("包名")

问题3: GitHub包安装失败
----------------------------------------
错误: remotes::install_github() 失败

解决方案:
1. 检查网络连接
2. 使用VPN
3. 或跳过高级可视化包

问题4: 内存不足
----------------------------------------
错误: "cannot allocate vector of size"

解决方案:
# Windows
memory.limit(size = 32000)  # 32 GB

# 或减少数据量进行测试
# 在QC步骤后采样5000个细胞

问题5: 权限问题
----------------------------------------
错误: "cannot create file"

解决方案:
# Windows: 以管理员身份运行RStudio
# Linux/Mac: 检查目录权限
chmod 755 /path/to/directory

📊 性能优化建议
================================================================================

1. 内存优化:
   - 关闭其他程序
   - 使用64位R
   - 增加虚拟内存

2. 速度优化:
   - 使用多核处理（如果支持）
   - 减少n_features参数
   - 使用SSD硬盘

3. 大数据集处理:
   - 使用断点续传功能
   - 分批处理样本
   - 使用服务器或云计算
